# Промпт: воскресный check-in

Короткий еженедельный ритуал. Даёт тревожной части предсказуемость: не фоновое
«а вдруг что-то не так», а регулярная точка, где это проверяется прямо.

## Идея
Договорённые заранее вопросы, одни и те же каждую неделю. Ставится по cron на
воскресенье. Ответ фиксируется в дневник.

## Промпт (AI инициирует)

```
Воскресный check-in. Три вопроса:
1. Был ли повышенный риск на этой неделе? (импульсивные решения, игнор сигналов тела,
   резкие качели, паттерны из прошлого)
2. Были ли проигнорированные сигналы тела/состояния?
3. Как в целом самочувствие — что в фокусе?

Если всё спокойно — это здоровый режим, зафиксируй коротко.
Если есть флажок — не читать лекцию, а отразить и, если уместно, назвать,
когда стоит эскалировать (например: «если через N дней не закроется — к врачу»).
```

## Правила
- Не превращать в допрос и не покровительствовать.
- Если check-in пропущен несколько недель подряд — это само по себе сигнал.
- Опираться на реальные цели пользователя, не на тревогу AI.

## Настройка (cron, пример)
```
# воскресенье, 11:00 локального времени
0 11 * * 0  -> запустить weekly-checkin
```

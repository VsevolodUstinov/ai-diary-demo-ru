# Карта зон уверенности (упрощённо)

> Сильно упрощённая демо-версия. Идея: разные зоны жизни находятся на разном уровне
> уверенности. Задача — переносить уверенность из «прокачанных» зон в «отстающие».

## Работа / бизнес
- Уверенность высокая
- "На моих условиях"
- Автономия + интерес + делать настоящее
- Зона где давно много практики

## Друзья / социальное
- Уверенность растёт
- Могу собирать свой круг, создавать пространства (вайб-салоны)
- Учусь говорить "нет" без вины

## Тело / здоровье
- В работе
- Спорт, сон, питание — базы
- Паттерн "оттягивать заботу до дискомфорта"

## Отношения / life partner
- Наименее уверенная зона
- Конфликт: Автономия ("на моих условиях") vs близость
- Задача: перенести уверенность из работы/дружбы сюда — через "получается", а не через
  ожидание готовности

## Мета-принцип
- Уверенность не врождённая — накапливается практикой
- В работе понадобились годы → в отношениях тоже нужна практика, а не сразу "мастерство"
- Переносить навык "разберёмся" из прокачанных зон в отстающие

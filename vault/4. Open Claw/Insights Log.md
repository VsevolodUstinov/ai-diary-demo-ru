# Insights Log

Ключевые инсайты из терапии, дневников и рефлексии. Компактный индекс (≤8000 символов,
чтобы auto-loading не обрезал). Полное описание каждого инсайта — в [[Insights]] как
отдельный файл.

> Подчищенная демо-версия: показаны несколько инсайтов из реального лога. Самые
> чувствительные (конкретные люди, отношения, зависимости) убраны.

**Как пользоваться:** найди по дате/названию, открой по wiki-link для деталей и
дословных цитат.

## Ранние (прорыв)

- **[[2026-01-01-child-not-guilty|Ребёнок не виноват]]** — главный прорыв. Ложное
  основание всей конструкции: "меня наказали → я плохой".
- **[[2026-01-29-freeze-safety-part-defense|Замирание — защита Safety Part]]** —
  реакция на стресс = замирание, кроме "полной жопы".
- **[[2026-01-31-trickster-rehabilitated|Trickster реабилитирован]]** — "не помню
  чтобы трикстер делал что-либо плохого за последние много лет". Наказание закончено.
- **[[2026-01-31-root-loss-of-meaning|Корень — потеря смысла, не травмы]]** — нужен
  рычаг (leverage). "Либо ярко, либо никак."

  можно медленно; если энергичный — быстро и весело."

  "всю жизнь было грустно". С грустью *рядом*, но не *вместе*.

  выбираю насколько интенсивно чувствовать".

# Архитектура и как пересобрать аналог

## Стек

| Слой | Что | Роль |
|---|---|---|
| Хранилище | **Obsidian** (или любой markdown-фолдер) | дневник, карты, инсайты, транскрипты |
| Агент | **OpenClaw** (или аналог) | доступ к папке на чтение/запись, инструменты, cron |
| Контекст | bundle-скрипт | склейка всего корпуса в chunks, скользящее окно |
| Ритмы | cron | утренний апдейт, недельный check-in, напоминания |
| Вход | голос/текст + транскрипция сессий | первоисточник |

## Поток данных

```
Голос/текст в момент ─┐
                      ├─► Obsidian vault ──► bundle ──► AI-агент (полный контекст в окне)
Транскрипт сессии ────┘        │                              │
                               └──────── cron ────────────────┘
                        (утр. апдейт, check-in, напоминания)
```

## Ключевой трюк: «перезагрузи всё»

Наивно читать 250+ файлов по одному — дорого и нестабильно. Решение —
[`../scripts/diary-context-bundle.py`](../scripts/diary-context-bundle.py): склеивает
весь корпус в несколько кусков по ~250k символов, читаемых за ~3 команды `cat`.

**Скользящее окно** (чтобы влезать в лимит):
- последний полный + текущий месяц → **сырой текст** (дневники и сессии дословно);
- всё старше → **месячные саммари** (сжато, с цитатами и ссылками на источник);
- скрипт сам считает окно от сегодняшней даты.

## Порядок сборки (от каркаса к деталям)

1. Фреймы (модель мира, карта зон, ценности, `agents.md`)
2. Части (карта субличностей)
3. Инсайты (индекс + каждый инсайт + интенции)
4. Люди
5. Месячные саммари (старые месяцы, сжато)
6. Сырьё: сессии + дневники за последние месяцы, дословно, по датам

## Как повторить за 6 шагов

1. **Хранилище.** Obsidian/markdown-фолдер. Папки: дневники, карты, инсайты, люди, сессии.
2. **Агент.** OpenClaw (или аналог) с доступом к папке на чтение/запись.
3. **Роль.** Возьми [`../templates/agents.md`](../templates/agents.md), перепиши под себя:
   зеркало, антифейки, что нельзя трогать без подтверждения.
4. **Контекст.** Настрой bundle, чтобы «перезагрузи всё» реально клало весь корпус в окно.
5. **Ритмы.** cron на утренний апдейт, недельный check-in, напоминания.
6. **Транскрипты.** Если работаешь с терапевтом — заводи транскрипты сессий как первоисточник.

## Настройка bundle-скрипта

В [`../scripts/diary-context-bundle.py`](../scripts/diary-context-bundle.py) поправь:
- `VAULT` — путь к твоему дневнику;
- `OUT` — куда класть chunks;
- пути блоков (папки сессий, дневников, саммари) под свою структуру;
- `CHUNK_MAX` — под лимит tool-результата твоего агента.

Запуск: `python3 scripts/diary-context-bundle.py` → читаешь `chunk-NN.md` по порядку
через `cat`, список и порядок — в `MANIFEST.txt`.
